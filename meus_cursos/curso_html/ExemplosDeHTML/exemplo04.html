Links

Os links servem para nos ligar a outros documentos tanto para áreas dentro da mesma página, quanto para arquivos externos.

Links internos

Os links internos nos levam a uma outra área do mesmo documento. Para isso, o link precisa apontar para uma tag com determinado identificador.

Links para outros documentos

Os links para outros documentos, nos levam para uma página do mesmo site ou para endereços externos ao site atual.
Para isso, o link precisa apontar para o endereço do documento: o caminho para uma página, uma url, etc.
