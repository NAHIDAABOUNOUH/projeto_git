Seja bem-vindo(a) ao nosso terceiro exemplo de HTML acessível

Nesse encontro iremos construir a estrutura básica do HTML, além de aprender o que são atributos, listas, tags de ênfase e a instrução especial para o navegador.

Instrução especial DOCTYPE

A instrução especial DOCTYPE serve para informar o navegador qual a versão do HTML que iremos utilizar no documento.

Atributos

Atributos servem para atribuirmos uma característica para determinada tag.

Hello! This text is in english!

Listas

As listas servem para listar itens!

Listas ordenadas

As listas ordenadas são utilizadas quando a ordem dos itens é importante.
São necessárias duas tags para criar essa lista:
tag ol, e tag li.

Exemplo de lista ordenada com frutas favoritas

  1. Banana
  2. Maçã
  3. Uva

Listas não ordenadas

Utiliza-se listas não ordenadas quando a ordem dos elementos não é importante.
Essa lista precisa das tags: ul e li.

Exemplo de lista não ordenada com algumas tecnologias assistivas

  • Leitor de telas NVDA
  • Dosvox
  • Talkback
  • Track ball

Lista de definição

Utiliza-se lista de definição quando é necessário informar a definição de alguns elementos.
Essa lista precisa das tags: dl, dt e dd.

Exemplo de lista de definição com linguagens de programação

HTML
HTML não é uma linguagem de programação, mas sim de marcação de texto.
JavaScript
JavaScript é uma linguagem de programação que deixa as páginas web mais dinâmicas.
CSS
CSS serve para configurar a apresentação visual de uma página web.

Tags de ênfase

As tags de ênfase servem para destacar e dar um peso semântico maior para o texto.

Tag 'strong'

Essa tag deixa o texto semanticamente mais forte e em negrito.

Tag 'em'

Essa tag serve também para destacar um texto, dando ênfase e deixando as letras em itálico.
